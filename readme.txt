== Installation ==

Copy "social-referrals" folder to your WordPress Plugins Directory.
Activate plugin via WordPress settings.

== Other Notes ==

No shortcodes yet .. but it does keep track of the number of referrals in the last 24 hours.

Two WP options are set with these values and updated whenever a new referrer is logged.

social_referrals_facebook_count
social_referrals_twitter_count
